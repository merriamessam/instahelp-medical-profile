# 🩺 InstaHelp – Medical Emergency Profile

A responsive web application designed to provide instant access to a patient's critical medical information through a QR Code.

The public emergency page allows first responders to quickly view life-saving information, while authorized medical personnel can securely access the patient's complete medical history.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/merriamessam/instahelp-medical-profile.git
```

Go to the project directory

```bash
cd instahelp-medical-profile
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
